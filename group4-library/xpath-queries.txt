====================================================================
Group 4 - Library Management System
XPath Queries 
====================================================================

Context document: library.xml

--------------------------------------------------------------------
QUERY 1 — Count Total Books
--------------------------------------------------------------------
Expression:
count(/library/books/book)

Used in: library.xsl header stat (Books), book inventory
count pill, and sarms.xsl Total Books stat card.

--------------------------------------------------------------------
QUERY 2 — Count Total Borrowing Records
--------------------------------------------------------------------
Expression:
count(/library/borrowingRecords/record)

Used in: library.xsl header stat (Records) and records
count pill.

--------------------------------------------------------------------
QUERY 3 — Count Active Records
--------------------------------------------------------------------
Expression:
count(/library/borrowingRecords/record[status='Active'])

Used in: library.xsl header stat (Active) and sarms.xsl
Active Loans stat card.

--------------------------------------------------------------------
QUERY 4 — Count Returned Records
--------------------------------------------------------------------
Expression:
count(/library/borrowingRecords/record[status='Returned'])

Used in: library.xsl header stat (Returned).

--------------------------------------------------------------------
QUERY 5 — Count Overdue Records
--------------------------------------------------------------------
Expression:
count(/library/borrowingRecords/record[status='Overdue'])

Used in: library.xsl header stat (Overdue), overdue notices
count pill, and sarms.xsl Overdue stat card.

--------------------------------------------------------------------
QUERY 6 — Count Returned Late Records
--------------------------------------------------------------------
Expression:
count(/library/borrowingRecords/record[status='Returned Late'])

Used in: library.xsl header stat (Late).

--------------------------------------------------------------------
QUERY 7 — Select All Overdue Records
--------------------------------------------------------------------
Expression:
/library/borrowingRecords/record[status='Overdue']

Used in: library.xsl overdue notices section to generate
overdue notice cards.

--------------------------------------------------------------------
QUERY 8 — Sum Total Physical Copies
--------------------------------------------------------------------
Expression:
sum(/library/books/book/copies)

Used in: sarms.xsl Total Copies stat card.

--------------------------------------------------------------------
QUERY 9 — Count Books by Category (Dynamic)
--------------------------------------------------------------------
Expression:
count($libraryDoc//books/book[category = current()])

Used in: sarms.xsl Books by Category chart data, iterating
over each distinct category found in the collection.

--------------------------------------------------------------------
QUERY 10 — Borrowing Status Counts for Chart
--------------------------------------------------------------------
Expressions:
count($libraryDoc//borrowingRecords/record[status='Active'])
count($libraryDoc//borrowingRecords/record[status='Returned'])
count($libraryDoc//borrowingRecords/record[status='Overdue'])

Used in: sarms.xsl Borrowing Status pie chart data.

====================================================================
END OF XPATH QUERIES
====================================================================
